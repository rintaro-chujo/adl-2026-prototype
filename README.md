# 「モノにこころをあずけておりまして」展 — Poster Maker

科研費基盤B「モノの主観的価値のダイナミクスを基盤とした循環型経済のインタラクションデザイン」最終成果報告展の告知画像ジェネレーター。

共通のカラーパレットと文字組み（デザインシステム）は固定したまま、**それぞれが自分の「かたち」と「色」で告知画像を作れる**シングルページの Web アプリです。

## 使い方

- **ひらくたびに、ちがう一枚**: アクセスするたびに色・かたち・ぼかしがランダムな一枚が自動生成されます（`?seed=…` を付けると同じ一枚を再現）。
- **じどうで つくる**: かたちの出どころを「おまかせ／ぐうぜん（抽象ブロブ）／ものの形（マグカップ・傘・鍵など8種の抽象シルエット）」から選択。「あたらしく つくる」で引き直し。
- **うごかす**: 色・ぼかし・かたちが少しずつ変わり続ける自動再生モード。タップで止まります。手描きのかたちもそのまま揺らせます。
- **うごきを GIF で保存**: 再生中のうにょうにょをシームレスループの GIF（480px・約3.8秒）でダウンロード。エンコーダは依存なしの純 JS 実装。
- キャンバスをマウス／タッチでなぞると、なぞった軌跡が図形になります。
- なぞり終えると図形が自ずと **ぼやけて** いき、好きなところで画面をタップすると止まります（スライダーでも調整可）。
- **描き方**: 面（塗り）／ 線 を切り替え。線モードには太さ調整あり。
- **塗り**: かたちに色 ／ 背景に色。
- **いろ**: 珊瑚・空・桜・陽・若草・藤・墨 の 7 色。
- **サイズ**: ポスター縦（A/B 比）／ Instagram 4:5。
- 「画像を書き出す（PNG）」で原寸解像度の画像を保存できます（画面＝出力）。

## 動的ポスター PNG（/poster.png）

Next.js の Route Handler が、**アクセスするたびに違うポスター PNG** をサーバ側で生成します（SVG 組み立て → resvg でラスタライズ、`Cache-Control: no-store`）。OGP 画像にもこのエンドポイントを使っています。

クエリ:

| パラメータ | 値 | 説明 |
|---|---|---|
| `seed` | 任意の文字列 | 同じ一枚を再現（省略時は毎回ランダム） |
| `shape` | `auto` / `blob` / `mono` | かたちの出どころ（既定 `auto`） |
| `size` | `poster` / `insta` | 文字組み（既定 `poster`） |
| `w` | 320–2400 | 出力幅 px（既定 1200） |

例: `/poster.png?shape=mono&size=insta&w=1080`

フォントはサーバ描画用に `assets/fonts/`（Cormorant Infant / Shippori Mincho のサブセット TTF）を同梱。文字組み SVG の tspan 絶対座標は `textLength` で補正しています。

## 開発

アプリ本体はビルド不要の静的 HTML 1 枚（`index.html`）。ローカルで開くだけで動きます（Web フォント読み込みにネット接続が必要）。GitHub Pages でもそのまま公開できます。

`/poster.png` を含むフル構成は Next.js:

```bash
npm install
npm run dev    # index.html（/）+ /poster.png を localhost:3000 で
npm run build  # 本番ビルド（Vercel はこれを実行）
```

`npm run sync` が `index.html` と `assets/` を `public/` にコピーします（`public/` は生成物なのでコミットしない）。デプロイ先ドメインが決まったら `index.html` の `og:image` の絶対 URL を差し替えてください。
